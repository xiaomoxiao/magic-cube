1.介绍
利用opencv自带的svm算法，进行魔方颜色的识别，识别的效果取决于选取样本数量的多少，以及在当时光强下采集的照片的质量。
对于样本中反光较为严重，对数据影响较大时一定要删除此数据。

2.使用方法
